# Домашнее задание

Написать функции декораторы, которые позволят собрать бутерброд.

Создайте функцию make_sandwich, которая ничего не будет делать
Создайте функции декотораторы, которые отвечают за вывод ингредиентов в стандартный поток вывода 
Оберните функцию make_sandwich созданными декораторами

В результате запуска программы, в терминал должна быть напечатана следующая строка

```
Bread
Salat
Tomato
Meat
Bread
```
