# Домашнее задание

Написать функции декораторы, которые позволят собрать бутерброд.

Создайте функцию make_sandwich, которая возвращает пустую строку
Создайте функции декотораторы, которые отвечают за "наполнение" результата выполнения функции make_sandwich ингредиентами
Оберните функцию make_sandwich созданными декораторами.
Выведите результат вызова функции make_sandwich в терминал

В результате запуска программы, в терминал должна быть напечатана следующая строка

```
Bread
Salat
Tomato
Meat
Bread
```
