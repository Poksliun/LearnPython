# Домашнее задание

Напишите собственную реализацию кеша в виде декоратора.

Декоратор cache должен сохранять информацию о результате выполнения функции и возвращать результат без вызова функции,
которую он задекорировал, если функция ранее вызывалась с такими же аргументами.

Для хранения результатов рекомендуется использовать словарь.
В качестве первой версии декоратор может не учитывать именованные аргументы функции, а кешировать только по позиционным.
